# Experiments log

Append one entry per training run or architectural change. Format:

```
## YYYY-MM-DD — short title
- Hypothesis:
- Change vs prior run:
- Setup: env, model size, training steps, seed
- Result: numbers + plots (link)
- Takeaway / next:
```

Goal: avoid the "doom loop" of forgetting what we already tried and silently repeating dead ends.
